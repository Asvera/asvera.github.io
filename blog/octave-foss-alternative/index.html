Ocatve FOSS alternative to MATLab

2024-04-20 | FOSS_tool

Being an engineer with non CS/IT majors comes with learning software rather building one. There are various softwares for designing,ploting,simulation normally all these comes with the same pakaged software such as Matlab, GNUplot, Fusion360, Blender, Octave, FREECAD,etc...

For this blog, lets only talk about Plotting software such as Matlab, GNUplot, Octave.

The most popular and commanly used in university and collage is none other than Matlab. It's mostly used by branches related Electrical and Mechanical Engineering.Rather, than saying Matlab is software saying it's an programming Language will be more accurate.

There are 2 different things:

  • Matlab software
  • Matlab Programming Language

In Engineering syllabus there course control Engineering it could rather be whole new specialization. In this course use various methods such as Bode plot, Nyquist Plot, Root Locus, etc.. to check the stability of a system. Since, I'm FOSS supporter how can is use MATLab thus i was using Ocatve.

Ocatve is and open source software written with C++ intended for carrying out numerical calculations and simulations with same implementation or syntax as Matlab language. Other difference being Open Source it use and Modularize approach to load pakages.

This Modularize appproach makes it utilize fewer resources
